# Java Fundamentals Interview Questions

## Q1. What is Java?

Java is a high-level, object-oriented programming language widely used for backend and enterprise applications.

---

## Q2. What is JVM?

JVM (Java Virtual Machine) executes Java bytecode and provides platform independence.

---

## Q3. Difference between JDK, JRE, and JVM?

- JVM executes Java programs
- JRE provides runtime environment
- JDK provides development tools
