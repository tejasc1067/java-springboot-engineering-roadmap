# Common Mistakes in Java Fundamentals

## 1. Forgetting semicolon

Incorrect:

System.out.println("Hello")

Correct:

System.out.println("Hello");

---

## 2. Wrong main method signature

Correct:

public static void main(String[] args)

---

## 3. Case sensitivity

Java is case-sensitive.

Incorrect:
system.out.println()

Correct:
System.out.println()
