# Java Fundamentals

## Overview

This section is designed for complete beginners who are starting their Java and backend engineering journey.

The goal is to build strong programming fundamentals before moving into advanced Java, Spring Boot, and backend development.

---

# Topics Covered

- Introduction to Java
- JDK vs JRE vs JVM
- Structure of Java Program
- Variables and Data Types
- Operators
- Conditional Statements
- Loops
- Arrays
- Methods
- Memory Basics
- Compilation and Execution Flow

---

# Learning Goals

After completing this section, you should be able to:

✅ Understand how Java programs run  
✅ Write simple Java programs  
✅ Understand variables, methods, loops, and arrays  
✅ Understand basic memory concepts  
✅ Read and debug beginner Java code
