# Stack vs Heap Diagram Notes

Create a beginner-friendly diagram showing:

STACK MEMORY:
- method calls
- local variables
- references

HEAP MEMORY:
- objects
- arrays
- String objects

Example Flow:
int age = 25;
String name = "Tejas";

Show:
- age stored directly in stack
- name reference in stack
- actual String object in heap

Recommended Tools:
- Excalidraw
- Draw.io
