# Java Fundamentals Theory

---

# 1. What is Java?

Java is a high-level, object-oriented programming language used to build:

- Backend applications
- Enterprise systems
- Android apps
- Cloud applications
- Microservices

Java is platform independent because of the JVM (Java Virtual Machine).

---

# 2. Why Java is Popular?

Java is widely used because:

- Platform independent
- Secure
- Scalable
- Object-oriented
- Huge ecosystem
- Strong community support

Java is heavily used in backend engineering and enterprise applications.

---

# 3. JDK vs JRE vs JVM

## JVM (Java Virtual Machine)

JVM executes Java bytecode.

It provides:
- Memory management
- Garbage collection
- Platform independence

## JRE (Java Runtime Environment)

JRE contains:
- JVM
- Required libraries to run Java applications

## JDK (Java Development Kit)

JDK contains:
- JRE
- Compiler
- Development tools

JDK is used for developing Java applications.
