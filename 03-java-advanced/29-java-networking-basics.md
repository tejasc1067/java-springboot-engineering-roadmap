# TCP Overview

- reliable communication
- ordered delivery
- connection-oriented communication

- HTTP
- databases
- APIs
- microservices

Very important backend networking protocol.

# UDP Overview

UDP provides:
- lightweight communication
- lower overhead
- faster transmission

BUT:
- no guaranteed delivery

Very important networking tradeoff.
