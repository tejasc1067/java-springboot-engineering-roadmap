# tdd-demo

A `ShippingFeeCalculator` built test-first. What's committed here is the final green state; the topic 11 markdown walks through the red-green-refactor cycles that produced it.

Run the tests:

```
mvn test
```

To *feel* TDD, recreate it: delete `ShippingFeeCalculator.java`, then bring back one test at a time, watching it fail (red) before you write the code that makes it pass (green).
